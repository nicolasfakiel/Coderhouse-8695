Quiénes somos

Doc Advisor es la primera plataforma en la que podés leer y escribir valoraciones sobre profesionales de la salud.

Sin embargo, Doc Advisor no es un proyecto real (por ahora). Es el proyecto final del curso de Javascript de Coderhouse.

El sitio fue creado usando estos lenguajes y Frameworks:
  • ☆ HTML
  • ☆ CSS
  • ☆ Materialize CSS
  • ☆ Javascript
  • ☆ Local y Session Storage
  • ☆ JQuery
  • ☆ Json
  • ☆ AJAX

El proyecto fue creado por Nicolás Fakiel. Si me querés contactar podés hacerlo por LinkedIn.
